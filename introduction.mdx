# Mega CLI

A powerful command-line interface that streamlines local development and interaction with MegaETH testnet and mainnet. Whether you're a smart contract developer, frontend engineer, full-stack developer, or user, Mega CLI provides one consistent workflow across networks.

![Mega CLI Banner](/images/banner.png)

## What is Mega CLI?

Mega CLI wraps foundational blockchain development tools like viem and Foundry (Anvil, Forge, Cast, and Chisel) with an easy-to-use interface specifically configured for the MegaETH ecosystem.

## Why Use Mega CLI?
- **Streamlined Interactions**: Quickly perform essential tasks like checking balances and requesting testnet tokens with just your keyboard
- **Simplified Workflow**: Reduce boilerplate setup and configuration time
- **Developer Experience**: Intuitive commands designed for devs
- **Network Aware**: Select local, MegaETH testnet, or MegaETH mainnet with a consistent `--network` option
- **Custom RPCs**: Override default endpoints safely with chain ID validation
